#  Clinic Chain

##  Problem
Healthcare records are often fragmented, especially in small and rural clinics. Patients frequently lose their medical history, leading to repeated tests, delayed treatment, and poor continuity of care.

## Solution
Clinic Chain is a centralized digital health record system that allows patient data to be securely stored and accessed across multiple clinics. Using a unique patient ID and QR-based access, doctors can instantly retrieve medical history anytime, anywhere.

##  Features
- 🔐 Secure authentication (JWT)
- 📷 QR-based patient record access
- 🏥 Multi-clinic data sharing
- 📁 File upload for medical records
- 📊 Simple dashboard for clinics

## Tech Stack
- Frontend: HTML, CSS, JavaScript, EJS
- Backend: Node.js, Express.js
- Database: MongoDB
